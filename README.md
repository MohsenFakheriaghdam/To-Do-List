# To-Do-List 📝

A simple and efficient application for managing your daily tasks

## 🎯 Features

- ✅ Add new tasks
- ✏️ Edit tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed
- 💾 Auto-save functionality
- 🎨 Clean and attractive user interface

## 🚀 Quick Start

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Installation & Usage

1. Clone the repository:
```bash
git clone https://github.com/MohsenFakheriaghdam/To-Do-List.git
cd To-Do-List
```

2. Open `index.html` in your browser or use a local server

3. Start adding your tasks!

## 📁 Project Structure

```
To-Do-List/
├── public/          # Public files
├── src/             # Source code
├── assets/          # Images and resources
└── index.html       # Main page
```

## 🛠️ Technologies Used

- **HTML5** - Page structure
- **CSS3** - Styling and design
- **JavaScript** - Application logic

## 💡 How to Use

1. **Add Task**: Enter text and click the add button
2. **Complete Task**: Click on a task to mark it as completed
3. **Delete Task**: Click the delete button for each task
4. **Edit Task**: Click the edit button and modify the text

## 📱 Compatibility

The app works on all devices:
- 🖥️ Desktop
- 📱 Mobile phones
- 📲 Tablets
